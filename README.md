play-react-template
=====

My project template with Play2 + React.js

Installation
=====
```
npm install -g create-react-app
cd front
npm install
npm run build
```

Run locally
=====
```
sbt run

cd front
# build js once
npm run build

# auto build
npm run watch:build

# auto testing
npm run test
```

distribution
=====

```
sbt dist
```
